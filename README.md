# My Issues


## 프로젝트 소개, 링크

- 깃헙의 레포를 저장하고, 저장한 레포의 이슈를 모아볼 수 있는 사이트입니다.  
- 깃헙 API의 시간당 호출량이 60번이라, 참고 부탁드립니다.🙏
- 링크: https://master--beautiful-marzipan-2e504b.netlify.app/




## 프로젝트 시작

```
yarn start
```

## 사용된 라이브러리
- Craco - CRA 설정
- Tailwind, Heroicons - 스타일링


## 프로젝트 구조
- apis
- components
- constants: 상수 관리
- context: 상태를 관리하는 context
- hooks: 커스텀 훅
- pages
- routes
- utils: 유틸함수들을 모아둔 곳

## 구현 기능
- 레포 검색 기능
- 레포 저장, 삭제 기능 (4개까지 가능)
- Local Storage에 저장
- 이슈 모아보기 (+ 페이지네이션, 상태 필터)
